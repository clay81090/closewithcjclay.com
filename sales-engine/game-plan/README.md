# Game Plan — Call Prep Files

One folder per prospect. Use on Closer / enrollment calls.

## Folder structure

```
sales-engine/game-plan/
  {prospect-slug}/
    ONE-CALL-CLOSE-SCRIPT.md   ← pull up on the call (word-for-word)
    CALL-BRIEF.md              ← 30-second pre-call scan
    SUMMARY-SNAPSHOT.md        ← full buyer read + context
    SAM-SETTER-NOTES.md        ← optional, when setter call exists
```
